# Orgs_upload_app
此项目为信息上传认证功能，旨在学习通过华为云提供的区块链服务进行HyperLedger Fabric 项目开发

可以直接在华为云上创建组织，通道，节点以及链码的安装和实例化

文件夹证书&SDK为华为云下载的证书和sdk配置文件
